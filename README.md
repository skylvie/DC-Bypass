# DC Bypass
Automatically bypass [DoubleCounter](https://doublecounter.gg/) verification via a Chrome extension

## Setup
1. Code -> "Download ZIP"
    1. Extract ZIP
2. Go to `chrome://extensions`
    1. Enable "Developer mode"
    2. Click "Load unpacked"
    3. Pick the extracted ZIP archive
3. gg

## How?
DC is useless and will literally accept a forged `X-Forward-For` header
